# BoneVPython

Create a bone with vPython 

# How to use

**Create a instance with your size**

    osso = Bone(12)
    
**Define your vector with length, height and width**

    osso.setVector(1, 0, 0)
    
**And generate**

    osso.generate()
    
*See an [example](sample.py)*
